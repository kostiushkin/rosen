Description

Main module for handling collisions and writing collision related stuff.

This module triggers collision layer modules specified by user in rosen:start_link/1.

This is also a behaviour, since it can be used as the general part for implementing new collision layers.

Function Index

handle/2Triggers all layers, passing to every layer the output of the previous layer and the complete object list; the first layer receives two times the complete object list because there are no layers first.
